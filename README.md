## Purpose 
The Google books search app is a Google Books API search engine built with Apollo Server GraphQL API. The app was built using the MERN stack, with a React front end, MongoDB databse, and Node.js/Express.js server and API. The App allows users to create an account and save book searches to the back end. The app has an Apollo Server to use GraphQL queries and mutations to fetch and modify data in place of a RESTful API.

## Technologies-Used

## Getting-started
To run the application clone the code and from the root of the project directory run `npm run develop`. This way you are able to run both the client and server side on a single terminal and view the fully functioning app in the browser.

## Deployed-app
